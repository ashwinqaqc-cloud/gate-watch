🔥 Firebase Integration Guide

Follow these steps to enable real-time multi-user sync so all operators see live updates instantly with no lag.

STEP 1 — Create Firebase Project
1. Go to console.firebase.google.com
2. Click Add project → name it "gatewatch-buhasa"
3. Disable Google Analytics (not needed) → Create project
STEP 2 — Enable Realtime Database
1. Left menu → Build → Realtime Database
2. Click Create Database
3. Choose region: us-central1 (lowest latency globally)
4. Start in locked mode → Enable
STEP 3 — Set Security Rules
In Realtime Database → Rules tab, paste:
{ "rules": { ".read": "auth != null", ".write": "auth != null" } } (or use true for both during testing, then lock down later)
STEP 4 — Get Your Config Keys
1. Project Settings (⚙ gear icon) → General
2. Scroll to Your apps → Add Web App (</>)
3. Register app → copy the firebaseConfig object
STEP 5 — Update This HTML File
Open this HTML file in a text editor, find the line:
// FIREBASE_CONFIG_PLACEHOLDER Replace it with your actual config:
const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "gatewatch-buhasa.firebaseapp.com", databaseURL: "https://gatewatch-buhasa-default-rtdb.firebaseio.com", projectId: "gatewatch-buhasa", storageBucket: "gatewatch-buhasa.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" }; Then find:
// FIREBASE_ENABLE = false Change false to true
STEP 6 — Host on Firebase Hosting
1. Install: npm install -g firebase-tools
2. Login: firebase login
3. Init: firebase init hosting
4. Put this HTML in the public/ folder as index.html
5. Deploy: firebase deploy
6. Share the URL (e.g. gatewatch-buhasa.web.app) with all operators
⚡ What Firebase Gives You
✅ Real-time sync — entry by Gate 1 operator appears on Gate 2 in <100ms
✅ Multiple operators — unlimited concurrent users, no conflicts
✅ Offline support — works even without internet, syncs when reconnected
✅ Free tier — 1 GB storage, 10 GB/month transfer (more than enough)
✅ No server needed — just this single HTML file hosted anywhere
✅ Mobile optimised — works on any Android/iPhone browser
